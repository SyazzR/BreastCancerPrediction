# Breast Cancer Prediction v3
Breast Cancer Prediction using Support Vector Machine(ML) using Flask, TailwindCSS and PHP.


## Requirement
* NodeJS: min.v18
* NPM: min.v8
* TailwindCSS v3
* php MySQL

### Start
Install packages
```
npm install
```

### CSS
Run script
```
npm run dev
```
edit file `src/input.css`, file will be auto compiled into `dist/output.css`
